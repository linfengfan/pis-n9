一、鳄鱼体系的核心定位
稳健型 + 趋势型 + 风控型= 不赌、不追高、不硬扛= 靠 “低估 + 拐点 + 流动性 + 仓位管理” 赚钱= 心理稳定 + 组合抗风险 放在第一位
二、投资框架（三层闭环）
流动性（市场水位）
决定能不能买、买多少、敢不敢重仓。
公司研究（基本面）
决定买谁、拿不拿得住。
外部环境（政策 / 行业 / 监管）
决定方向、风口、风险。
→ 三者同时满足，才出手；缺一个，就观望。
三、核心策略（4 条铁律）
1. 明显低估 → 大量买
（价值底、安全垫足够 → 重仓）
2. 左侧标的 → 持续观望，拐点出现再重点买
（不提前抄底，不猜底，只做右侧确认）
3. 长期底仓 → 暴涨就卖（看流动性）
（底仓是用来拿的，但暴涨必须兑现利润）
4. 左侧谨慎，尽量买右侧
（只做顺势，不做逆势；只做确认，不做预判）
四、交易的真正意义
交易 ≠ 为了赚差价
交易 = 三个目的：
控制仓位（永远不被动）
心理正向引导（保持心态舒服、不焦虑）
增强组合抗风险能力（让账户更稳）
→ 哪怕不赚钱，只要能让心态稳、仓位稳、组合稳，交易就是对的。
五、鳄鱼体系的精华总结
看流动性、看基本面、看环境；
低估重仓、拐点买入、暴涨兑现；
交易不为赚快钱，只为心态稳、仓位稳、组合稳。
